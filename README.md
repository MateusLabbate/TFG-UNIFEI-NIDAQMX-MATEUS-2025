# Sistema para Aquisição e Análise de Distúrbios da Qualidade da Energia Elétrica

Este repositório contém os códigos-fonte e planilhas desenvolvidos para o Trabalho Final de Graduação (TFG) do curso de Engenharia Elétrica da Universidade Federal de Itajubá (UNIFEI).

O projeto consiste num sistema de medição e análise de fenômenos da Qualidade de Energia Elétrica (QEE), utilizando Python e hardware de aquisição de dados (![nidaqmx](https://github.com/ni/nidaqmx-python)) para monitorar distúrbios como Variações de Tensão de Curta Duração (VTCDs), distorções harmônicas e variações de tensão de longa duração.
